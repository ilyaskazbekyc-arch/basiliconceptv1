# Basilic.kz redesign prototype

Интерактивный React/Vite-прототип редизайна сайта Basilic.kz.

## Запуск локально

```bash
npm install
npm run dev
```

После запуска откройте локальный адрес, который покажет Vite.

## Сборка

```bash
npm run build
```

Проект подготовлен для публикации через GitHub Pages.
